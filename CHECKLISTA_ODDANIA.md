# Checklista oddania projektu

## Przed wysłaniem

- [ ] `npm install`
- [ ] `npm run dev`
- [ ] `npm run lint`
- [ ] `npm run build`
- [ ] repozytorium GitHub utworzone
- [ ] link do repozytorium uzupełniony
- [ ] ewentualny krótki filmik działania przygotowany

## Linki

Repozytorium GitHub:

```text
tutaj wkleić link do repozytorium
```

Filmik działania, jeżeli będzie wymagany:

```text
tutaj wkleić link do filmiku
```

## Notatki techniczne

- Projekt korzysta z React + Vite.
- `.gitignore` wyklucza `node_modules`, `dist`, `.env` i `.cache`.
- Jeżeli `npm run build` nie powiedzie się na Windows przez błąd `esbuild` / `EPERM`, należy opisać to jako problem środowiskowy i nie zmieniać kodu aplikacji tylko w celu obejścia tego błędu.
